Custom ROM Manager

Custom ROM Manager finds all the ROMs, GApps and other zip files that are downloaded on your device, lets you install or manage them easily. It also helps you discover and visit official sites of some ROMs/GApps from where you can download them or see changelogs. Visit the Play Store link to know more.

Privacy Policy

This page contains about what data does Custom ROM Manager uses, collects and shares. Please take your time to go through this section and know more about the same. By using Custom ROM Manager, you allow and agree to this usage of information, although the app uses as minimum information as necessay to function

What data does Custom ROM Manager use?
Custom ROM Manager works with the files (.zip files specifically) present on your device. As it targets Android O, it is required to ask storage permissions on Android 6.0 and above, and it can access files depending on whether storage permissions are available or not. It is highly recommended to grant storage permissions because without it, app is basically useless. It is also guaranteed that the app does not modify/write user files without his/her consent. Additionally, the app uses root access to write the recovery script used for flashing (which is done only with the user's consent), and root is also used for some file management operations on memory card as Android Framework does not provide a straightforward way to operate on them.
What data does Custom ROM Manager collect?
Custom ROM Manager only collects crash reports, analytics data based on app usage. This is done to fix bugs and improve UX. The collected data is only used internally and not shared whatsoever. It is also guaranteed that the app does not collect any data outside the scope of the app whatsoever.
What data does Custom ROM Manager share?
Custom ROM Manager does not share any data it collects to anybody else. The data collected is only used internally solely for the purpose of improving app.

Custom ROM Manager highly respects the privacy of the users, and it only uses the data it needs to function and improve.
