Custom ROM Manager

Custom ROM Manager finds all the ROMs, GApps and other zip files that are downloaded on your device, lets you install or manage them easily. It also helps you discover and visit official sites of some ROMs/GApps from where you can download them or see changelogs. Visit the Play Store link to know more.

This page serves as download page of Custom ROM Manager (free version). Starting with version 5.0, there will separate apks for kitkat (pre-lollipop) and lollipop (and later), since it may not be possible for an user to access specific version, hence this page exists, where you can find and download all variants available for Custom ROM Manager.

Help me choose

Select the correct options based on your device, and then hit download. If you are unsure about any option, select "Not sure". Do note that Custom ROM Manager is only supported from Android 4.4 and above, and the platforms supported are arm, arm64, x86 and x86_64.

All variants

Here you can find all the available variants of Custom ROM Manager. If you are installing Custom ROM Manager (version 5.0 or above, any release type) from Play Store, you need not worry about these variants as Google Play will automatically detect the appropriate variant for your device and install it.

Naming convention

custom-rom-manager-[version]-free-[release type]-[android version]-[platform].apk

  • Version - Version name (for example - 5.0)
  • Release type - Can be either beta or stable.
  • Android version - Minimum android version required for the variant to be able to run. (minAPI19 - Android 4.4 (kitkat), minAPI21 - Android 5.0 (Lollipop))
  • Platform - Platform supported by the variant. (Supported platforms are arm/arm64 and x86/x86_64. If you are not sure, select allABI)
